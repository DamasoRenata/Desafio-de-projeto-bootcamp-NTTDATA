O que são Tags?

Voltar

“Tag” é uma palavra da língua inglesa e quer dizer etiqueta. Na internet elas são usadas na organização de informações, agrupando as de mesma classificação para facilitar na busca de assuntos relacionados, podemos ainda dizer que são palavras-chave de busca.

Em HTML as tags também servem como etiqueta, porém com a finalidade de informar ao navegador que tipo de estrutura está sendo construída, tais como, títulos, parágrafos, imagens, links, entre outros. Aqui podemos dizer que Tags são os elementos estruturais de um blog, por exemplo.

Tags Head e Body

Voltar

As páginas HTML são divididas em dois elementos: head e body. No qual head significa cabeça e body, corpo.

A tag head é um elemento que contém informações sobre o documento que não são mostrados em tela, na página web. Ao contrário da tag body que já é aquela parte mostrada ao usuário em tela, como parte de uma página de um site. É nela que visualizamos os textos, imagens, links e vídeos.

Dentro destas tags podemos colocar outras tags, como de títulos, subtítulos, textos, imagens e links.

As Tags H

Voltar

São tags de títulos e subtítulos, utilizadas hierarquicamente, de h1 até h6, Sendo h1 o título principal que está, geralmente, na área head da página. Os demais colocamos no body seguindo a ordem de importância dentro do texto.

Tag P

Voltar

O elemento p representa um parágrafo e são usados para agrupar conteúdos relacionados de qualquer tipo, como textos, imagens e campos de um formulário. Dentro da tag p usamos outras tags de conteúdo para estilizar o texto, tais como itálico e negrito.

Tags de conteúdo

Voltar

As tags de texto definem textos, estilos de fonte, parágrafos, e afins. As mais básicas são b que transformam o texto em negrito, i em itálico.

Tags de listas

Voltar

Para criar listas, um índice, por exemplo, podemos utilizar a lista ordenada, ou não ordenada. Se quiser que seja numerada, a lista a fazer é a ordenada, a partir da tag ol. Mas se quiser uma lista apenas como tópico, a melhor a ser usada é a não ordenada, usando a tag ul. Posteriormente é necessário incluir dentro da lista a tag li para cada elemento da lista.
